busyram代表什么？

发表于 2017-08-02

最开始，我最好能够说明下为什么我的域名叫做busyram。这是一个很重要的问题。
ram是什么？

随机存取存储器（random access memory，RAM）又称作“随机存储器”，是与CPU直接交换数据的内部存储器，也叫主存(内存)。它可以随时读写，而且速度很快，通常作为操作系统或其他正在运行中的程序的临时数据存储媒介。

ram是智能设备里面最重要的东西，这样类比吧，CPU好似心脏，ram类似大脑临时存储记忆的海马体，rom就相当于长期记忆的大脑皮层。
busyram的意思，重点在busy和ram上，你的记忆、思维需要动起来。
没有用的ram，再多也没用，因为堆叠出来的无用东西就是浪费。如果你用iPhone的话，iPhone6还是1G的ram，此时其它的Android手机都是2G甚至4G，但是iPhone6并没有给人卡顿的感觉，Android就会让人觉得。
所以，我期望能够做一个busyram！

free ram is useless ram!
