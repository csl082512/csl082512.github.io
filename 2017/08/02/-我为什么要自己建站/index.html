我为什么要弄个人网站

发表于 2017-08-02

我的网站就是我的名片
有事没事都可以写一些东西
谁愿意看谁看，没关系
想写就写，不用在乎关系网络
真正属于自己的东西

我的选择：
Hexo静态博客模板+阿里云虚拟主机+阿里云域名购买

阿里云虚拟主机：
花了6块钱买了普惠版的一年先玩玩。

阿里云域名：
我购买了自己一直想注册的
www.busyram.com
后面可以自己玩了

Hexo这个比较麻烦，记录一下操作步骤，怕以后忘记了，也是学习的过程
步骤：

  • 安装Git
  • 安装node.js
  • 在node中执行：npm install -g hexo-cli。安装好Hexo的应用
  • 本地预览很简单，不说了
  • hexo关于配置到阿里云虚拟主机的几个问题和坑记录一下：

_config.yml这个配置文件：

#URL
url: http://bxu2713750775.my3w.com/myblog
root: /myblog/
permalink: :year/:month/:day/:title/
permalink_defaults:

【注】bxu2713750775.my3w.com 这个是域名，我申请的临时域名，待我的域名审核通过之后替换即可。这个设置不好的话，会导致资源加载有问题。
阿里云的资源分配是这样的，用户所有的文件都在以下htdocs文件夹下。
我自己在里面建了一个myblog，用于个人博客。

#Deployment

##Docs: https://hexo.io/docs/deployment.html
deploy:
type: ftpsync
host: bxu2713750775.my3w.com
user: **
pass: **
remote: /htdocs/myblog/
port: 21
ignore: .DS_Store
connections: 1
verbose: true
【注】这里，冒号之后有空格，切记。。。。。。被坑惨了，就差打客服电话了
具体解释看Hexo的官方文档吧，台湾小伙写的，还不错
