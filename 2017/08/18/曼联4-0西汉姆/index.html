英超第一轮：曼联4-0西汉姆

发表于 2017-08-18

好久没看球这么爽了
第一个进球，速度和角度
第二个进球，力量和选位
第三个进球，速度与技巧
第四个进球，流畅与爆发
一场球下来，一扫上赛季的阴霾。
有速度，有防守强度，有高度，有流畅配合，我都开始有点小兴奋了。
颤抖吧凡人们，期待看踢曼城了，红蓝之战，看将爆出什么样的火花。
看到了豌豆在场，确发现穿的是对方的球衣。看到豌豆被曾经的队友重点照顾，踢倒在地上，心里各种酸楚！
我专门看来埃弗顿的比赛，不为什么，只为鲁队长。鲁队长绝对的进攻核心，长传、调度、转移，进球，价值完全体现。相比之下，埃弗顿还缺一个强力中锋，那样估计就可以争四了！中后场很不错，前锋稍逊。
总之，曼联这个赛季可以联赛争冠，但是在欧洲赛场，不好说，和皇马还是有差距的。除了皇马，其它不足为惧！看好这个赛季！
曼联如果还缺什么的话，那就是缺一个左后卫了，希望肖能够找回自己！
