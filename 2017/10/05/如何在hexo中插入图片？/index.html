如何在hexo中插入图片？

发表于 2017-10-05

概述

之前一直懒得增加图片，现在打算好好补充下图片相关的。
按照知识的传递来说，信息量是这样的：
文字<图片<语言<视频
图片就有两种：静态图片和动态图片。

插入一个静态图片

插入图片的语法：
![name](links)
网络图片这次就不说，我说下如何记录本地图片。

  • 在source文件夹中增加img文件夹
  • 在img中添加要访问的图片
  • 通过相对路径访问：![name](/img/imgname)

测试一下：
![cat](/img/吉卜力cat.JPG)
