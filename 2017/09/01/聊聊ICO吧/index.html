聊聊ICO

发表于 2017-09-01

这次，我们聊下ICO吧。
ICO(initial coin offering)首次代币发行，提供了一种新的融资方式。
传统的融资方式：

  1. 私人注资
  2. 天使投资
  3. 银行借款
  4. 发行债券
  5. IPO

IPO是很多公司都想参与的，IPO之后发行的股份，流动性比较好，也能融到大量的资金。
但是，这些都有个前提，那就是准入要求，或多或少需要你提供良好的证明，注册资本、连续几年的营业收入，等等。
ICO就不一样，经常出现的场景是，只凭借一份几十页的白皮书，就能融资几个亿，略显疯狂。
我们来看看ICO的流程：

  1. 推出项目介绍网站，发布项目白皮书
  2. 开启ICO流程
  3. 结束ICO流程
  4. 披露信息

总体上，目前ICO的标的物可以分成以下几个种类：

  1. 虚拟商品类
  2. 收益权凭证类
  3. 基金份额类
  4. 公司股权类

这是一个比较快速的过程，很快就能够融资。这里面有几个比较重要的概念：
项目白皮书
项目白皮书是ICO项目的灵魂，一般阐述以下内容：

  • 项目简介
  • 市场概述
  • 任务和愿景
  • 我们提供的解决方案
  • 运作方式
  • 竞品分析
  • 发展规划
  • 管理规划
  • 代币介绍和发行计划
  • 代币的价值和分成计划
  • 资金用途
  • 团队和投资者

代币
代币一般是和现有的数字货币挂钩,数字货币一般是比较主流的，主要是比特币、莱特币、以太币、瑞波币等。设置一个比例，比如：500代币=1比特币。
代币一般基于以太坊的智能合约发行，也有一些平台自建。用户转入比特币，获取对应的代币。为了鼓励人参与，最先转入的人能够获得一些奖励。拿到了代币，象征着权益，后续可以转移交易。可以对接交易所，方便进行买卖。代币的价值是和项目的价值挂钩的，比如，如果项目发展很好，则估值很高，代币的交易金额就会快速上涨。或者，代币代表着某种权益，使用的权利，类似ripple币，则随着使用的人变多，价格也增加。因此，ICO更多可以称为产品众筹，结果取决于项目的表现。

举个例子：
我现在手头有10btc，按照现在的行情来看，我在OKCoin交易所可以变现310=30W。
目前，有个ICO项目，叫做hero，发行的代币为hc,标准为500hc=1btc,我是第一个投资的人，我转入了10btc，给了我50010=5000c，因为我是初始投资，又额外给了我500hc,于是我拿到了5500hc。hero通过发行代币，拿到了10btc，它可以在交易所变现30W，获得项目发展的初始资金。我可以一直持有这5500hc，类似股票一样。hero项目发展很好，hc币后面涨到了100元/hc,则我可以卖出，拿到5500*100=55W，投资收益25/30。

这个看起来ICO非常动人，融资非常迅速，17年上半年就融资了等值26亿人民币。这块属于监管的空白，我们看看监管的声音：

  1. 坚决反对和取缔
  2. 保持观望

我们拭目以待吧！
