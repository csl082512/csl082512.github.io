人的素质1

发表于 2018-02-17

新年第二天，去电影院看电影。
刚坐定，一对夫妇，带着两个孩子，坐在我左边，隔着一个人。
我还纳闷，孩子挺大的，两个位置如何坐下？
果然不出我所料，熊孩子后面开始了：

  1. 小孩子开始嚷着一个人坐，呵呵
  2. 电影期间，小孩子貌似不感兴趣，各种嚷着回家
  3. 小孩子开始放手机视频，声音很大，我很愤怒，和那个男的说，让孩子声音小一点。他和孩子的对话让我很惊讶：请把声音关小一点好吗？我当时就震惊了，这居然用请？换做我的孩子，我直接教育了。小孩子不听，他就自己关小一点，然后孩子就大哭起来。
  4. 途中小孩子好几次都说要回家，有两次都跑到座位中间，大声哭喊，然而两个人又抱回来了

人的素质，彰显得淋漓尽致！
和小孩子无关，主要是父母的问题，以后如果这个孩子也为人父母，估计也是类似。教养是一个人的，和贫富无关！
