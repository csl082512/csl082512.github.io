如何迁移你的hexo博客？

发表于 2017-10-05

我要把我的博客迁移到新的机器
记录下步骤
1.安装git
2.安装node js
检验安装的命令：
git --version
node -v
npm -v
3.安装hexo
npm install -g hexo-cli
npm install hexo --save
检查安装是否正确
hexo -v
4.拷贝文件夹到新PC，原来设置好的配置文件和文章可以不变
5.设置GIT的身份
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
6.hexo d部署，会提示输入github的密码，输入即可

PS：
补充一个问题
最近在用hexo搭建github pages时，遇到一个问题，hexo安装没错，也能成功运行。启动也没错。
就是不能访问。。

原因是：
你的电脑端口被占用了。
hexo默认的端口是4000，如果你的电脑安装了福昕阅读器，，就是他，没错，坑爹吧！！！！
启动hexo s 的时候，用这个命令，换一个端口。
hexo s -p 5000
换成5000端口，果断可以访问

支付和教育

发表于 2017-09-19

关于支付和教育

两种方式:
支付切入场景
场景切入支付

基于工具的平台应该是一个连接器，连接人与教育服务。目前针对大学生市场，最大的问题是脱节，学与用脱节，理论和实践脱节。因此，最好的机会就是打通这个GAP，增加用户的黏性。
两个端:
用户端，基于支付宝平台，拿到更多的用户数据，包括学校认证信息，专业认证信息，消费数据信息，这些都是用户画像的一种积累。再打通外部数据，比如用户图书馆借阅书籍的数据，用户日常关注点的数据，分析用户的行为，为用户推荐服务。
服务端:打造开放的平台，各种丰富的教育相关服务可以快速接入，支付宝提供方便的支付一体化服务。
这两端最重要的因素就是数据。用户端的数据和用户特征分析。服务端的内容，教育最重要的就是内容质量。

理想的场景:
用户注册，通过学校认证，获得用户画像，给用户推送教育服务。比如产品经理的课程，开发工程师的课程，考试课程，实习机会等等一揽子职业教育服务。用户选择服务，然后付款，享受线上和线下服务。用户大一大二，推送用户专业相关的行业信息，和毕业的学长进行互动。用户大三大四推送实习和工作机会。用户毕业，推送租房信息，推送置办物品信息，推送最新的职业服务。从而打造用户的黏性，升级为核心用户，和场景相关的。

提供一个平台，在校学生和已经毕业的学生可以沟通交流。弥补理论和现实的差距。人潜意识去寻找归宿感，圈子的认同感，在一个学校更亲切，切入场景更自然。无法做普通的人与人的社交，但是可以做基于事务和圈子的社交。

支付，首先打通校内的支付场景，一卡通充值，或者完全替代一卡通的电子一卡通。可以用于刷卡进图书馆，食堂，刷卡洗澡，购买水电，就医等，覆盖校内商家消费，校外商家消费，全场景。这个目前看来没有任何问题。关键是，如何创造更大的价值？给其它业务线带来用户和数据。

我们的目标，应该是让大学生更好的学习和生活！从而成为我们的忠实用户。

大学生市场的横向和纵向，横向覆盖教育、旅游、游戏、健身、饮食、美容等。纵向则由大学生向白领的过度。

徽行古道两日徒步

发表于 2017-09-17

16和17号两天是周末，正好和几个朋友一起组队去徽杭古道徒步。这是最初级的徒步首选，对于我这样的人来说，没啥难度，倒也合适。
徽杭古道处于浙江和安徽省交接，据说是当年徽商发源的地方。徽州穷，为谋生，只能走出群山，前往富饶之地经商。
本次选择反穿，即从浙江到安徽段，体验稍逊于正向穿越！
16号早上从上海出发，开车5个小时到达永来村，吃午餐开始爬山。我们抄了一个近道，从海拔600米爬到了1000米的蓝天凹。没有走南天门，而是从一个山谷穿越上去的。山路是当地的村民上山打山核桃踩出来的，沿途都是山核桃。临安小核桃据说很出名，但是正好现在还没上市，晚一两个星期就上市了。
山谷有溪水，水很冰凉。沿途遇到一个小屋，是两层的，砖石砌出来的，有一个小院，非常温馨。居住在这里，肯定非常安静。
到了蓝天凹，一个开阔的山坳口，都说从安徽过来会比较好，其中之一应该是到达蓝天凹的时候，一览群山的感觉。
9月中旬，已经初秋，水量不是很大，山上漫山遍野的茅草，显得非常柔和。
过了蓝天凹之后，就是一路的下坡路了，路还算好走，但是因为爬了很高的路，下坡的时候腿有点抖。一路都是山路，山路旁边就是溪水，很清爽。
快六点的时候，到了下雪堂，当晚在这里留宿，说实话，与我期待的有些出入，我期待的应该是有临水的小屋，红色的灯光，当地的特色菜。
洗了个热水澡，浑身通畅，然后开始吃晚饭，吃的不怎么样。吃完晚饭开始在楼顶燃起了篝火，放着音响，准备晚会。因为是山里，夜很黑，漫天的星星。我已经很久没有看到这么多星星了，让人觉得很舒服。山里温度较低，我换上了长袖长裤，开始等待烤羊排。
一群认识的朋友，在野外，燃起篝火，大家围着火谈论以前和现在，才是很棒的体验。这次来的因为一半的人不认识，大家玩的不是很嗨。
人在大城市里面待久了，总会怀念野外，大概就是人性吧。人毕竟是属于大自然的，而不是属于一堆个人创建的城市里。
休息了一晚，早上起来小腿微酸。我去了旁边的小溪边，发现水里面有条纹的鱼，黑色的条纹，貌似山泉里面的鱼都是这样的。我回头查查，这鱼是从哪里来的。
吃过早饭继续上路，沿途都是沿着山边缘开出来的小路，旁边是深深的峡谷，峡谷里面水不多，一半是因为是秋季，另一半是因为被大坝拦住了。我反而对一个草亭提起来了兴趣，草亭名唤“施茶亭“，以前为了给过往的路人小憩的。从岩石之中开凿出来，西面悬空，东面有一小庙。曾有楹联“索道萦回，重重水口。关山险阻，步步惊心“。茶与亭，总会让人暂时忘记疲惫，一盏茶，起身，继续赶路！
到了中午，就出山门了，吃过午饭，两日游结束！

教师节的犹豫

发表于 2017-09-11

昨天是教师节，按我一直的习惯，我会给我身边的老师发祝福的。但是，我竟然有些犹豫了，一直拖到晚上九点。
我到底是犹豫什么？
之前，我都是打电话或者发信息的，打电话有个问题，偶尔一次电话，有时候就会比较尴尬，尬聊，询问近况，等着某个契机，找到一些共同的话题。发短信会好一点，毕竟情绪都藏在了文字之下，等对方回复即可。
我犹豫电话还是信息。
一般，重要的事情我基本都打电话，基本不会通过信息的，毕竟信息意味着通知，不太适合。
我在微信上发了信息，给小学、初中、高中。和我高中三年的数学老师聊了一会，她对我一直很好！最后一次见面，好像是大一的寒假，那时一帮人特地去她家里感谢她，后面就再也没见过，每次回家都太匆匆！偶尔，看到微信我会点个赞，尽量选择一个比较合适的距离，小心翼翼地维护着。
和老师的关系，在什么时候都一样，不管怎样总是带着尊敬的。工作多年，再见老师都是饱含感情的，毕竟曾经的教导！
最近和老婆聊天，说到现在越来越多人不愿当老师了，付出和收入不成正比。这是一个病态的循环，教师现在的工资越来越低，工作量却越来越大。我姐也是高中老师，你羡慕她有寒暑假？想多了吧，她有时候连周末都没有，别说寒暑假了！早上要去早自习，很早的早自习，晚上要晚自习，结束了就很晚了，周末至少补课一天，剩下一天，暑假至少补课占据2/3，还需要改卷！然而，工资比普通公务员还低，呵呵，工作强度大很多！所以，公务员大家都挤破头去考，还是有它的原因的。
现在看来，唯独大学老师，应该会轻松一点，有自己的时间，可以做一些事情，虽然也有黑暗的地方，比如晋升制度，研究课题之类的，也就不说了，毕竟黑暗才是社会的真实规律！
不过，这么多老师中，我唯独没有记录任何一个大学老师的联系方式，不得不说有些遗憾。但是，也说明大学，老师和学生的关系总是那么脆弱，毕竟相处时间少，关系是需要时间培养的！

9月，吃石榴的季节！
每次老婆买回石榴，掰开，透出红的果粒，总让我不自觉想起高一的时候。那是一段很长的时间了，曾经15岁的时光，总会不自觉显现出来。
我的记忆倒退回去，高一，刚刚到学校报到，宿舍里面六个人，还没有全部认全。有一天，一个哥们拎回来了四五个拳头大的石榴回来，打开，一群人就开始横扫了，如此一来，大家之间的关系也就开始变的更加融洽了。
因此，每次吃到石榴，总会想起之前的时光，停留在记忆里面的那段时光。如今，大家已经各奔东西，估计是不会有机会再在一起吃石榴了吧。记忆之所以显得珍贵，很大程度上取决于它的不可倒退，那时、那景、那人，都无法再退回去了！
