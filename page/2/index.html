关于联系connection

发表于 2017-09-08

connection我称为联系，链接两个不同主体。
为了简单起见，我抽象了两个主体：人和物。
人和人的联系，称为社交，在这个领域腾讯占据领先地位。
人和物的联系，称为场景，这个领域有众多场景，淘宝、京东、点评等等。
物和物的联系，称为物联网，这个领域目前正在孵化，小米生态、百度占据。
支付是什么？支付是货币价值的流动。在人和人之间、在人和物之间、在物和物之间！
微信基于人与人的联系切入支付，水到渠成，支持收款、付款、转账等场景，同时支持部分人和物的场景。支付宝一直基于人和物的场景进行深化。微信和支付宝，两者相互竞争和渗透，微信切入越来越多人和物的场景，支付宝切入人和人的场景，结果现在已经显而易见。支付宝在人和人的场景上，完败！这是显而易见的，人和物本来就是弱关联关系。
我翻了下我的支付宝朋友记录：最近的一个朋友是一月份，是个同事，加我求敬业福，我没有。最新的朋友动态是五月，一个朋友分享了ofo红包的链接。我最新的一笔转账记录，是6月份，那个朋友说他微信没钱了，支付宝给我。社交关系差距还是蛮明显的。
我翻了下我最新的交易，早上我用支付宝付款码，付款早餐，领取了0.35元鼓励金。前一天缴了水电费，之前还有医院挂号的记录。
这样对比一下支付宝和微信：
人与人：微信绝对占优，支付宝没机会
人与物：支付宝绝对占优，微信有机会
物和物：双方都不占优，都有机会
上面就能够理解，为什么支付宝恐慌了，牌面处于劣势！微信，基于社交，主打社交，因此在人和物上，并没有像支付宝在人与人上一般发力！
因此，支付宝如果想打赢这场战争的话，就要充分利用自己的优势：深耕人与物，扩展物与物！

我们找了一家支付机构帮忙做资金托管，这是整个体系非常重要的点，真正做到T+0实时清算才是我们产品的亮点！各种服务都不错，但是有一点不好就是，价格太高了。有准入费用，有按年收取服务费。因为我们处于初期探索模式，因此收费这块我们比较谨慎，尤其是如何控制成本。
这个时候，团队的两个声音出来了：

  1. 收费太贵了，我们采用人工线下模式
  2. 继续使用服务，成本转嫁给商户
    对于1来说，很现实，采用低成本，但是这样实现，我们的产品的优势就不是那么明显了！对于2来说，采用服务，保证了我们产品特点，后面接入更多商户再扩展收费，但是有风险！

    钱该花的时候就要花，该投入就投入！

聊聊ICO

发表于 2017-09-01

这次，我们聊下ICO吧。
ICO(initial coin offering)首次代币发行，提供了一种新的融资方式。
传统的融资方式：

  1. 私人注资
  2. 天使投资
  3. 银行借款
  4. 发行债券
  5. IPO

IPO是很多公司都想参与的，IPO之后发行的股份，流动性比较好，也能融到大量的资金。
但是，这些都有个前提，那就是准入要求，或多或少需要你提供良好的证明，注册资本、连续几年的营业收入，等等。
ICO就不一样，经常出现的场景是，只凭借一份几十页的白皮书，就能融资几个亿，略显疯狂。
我们来看看ICO的流程：

  1. 推出项目介绍网站，发布项目白皮书
  2. 开启ICO流程
  3. 结束ICO流程
  4. 披露信息

总体上，目前ICO的标的物可以分成以下几个种类：

  1. 虚拟商品类
  2. 收益权凭证类
  3. 基金份额类
  4. 公司股权类

这是一个比较快速的过程，很快就能够融资。这里面有几个比较重要的概念：
项目白皮书
项目白皮书是ICO项目的灵魂，一般阐述以下内容：

  • 项目简介
  • 市场概述
  • 任务和愿景
  • 我们提供的解决方案
  • 运作方式
  • 竞品分析
  • 发展规划
  • 管理规划
  • 代币介绍和发行计划
  • 代币的价值和分成计划
  • 资金用途
  • 团队和投资者

代币
代币一般是和现有的数字货币挂钩,数字货币一般是比较主流的，主要是比特币、莱特币、以太币、瑞波币等。设置一个比例，比如：500代币=1比特币。
代币一般基于以太坊的智能合约发行，也有一些平台自建。用户转入比特币，获取对应的代币。为了鼓励人参与，最先转入的人能够获得一些奖励。拿到了代币，象征着权益，后续可以转移交易。可以对接交易所，方便进行买卖。代币的价值是和项目的价值挂钩的，比如，如果项目发展很好，则估值很高，代币的交易金额就会快速上涨。或者，代币代表着某种权益，使用的权利，类似ripple币，则随着使用的人变多，价格也增加。因此，ICO更多可以称为产品众筹，结果取决于项目的表现。

举个例子：
我现在手头有10btc，按照现在的行情来看，我在OKCoin交易所可以变现310=30W。
目前，有个ICO项目，叫做hero，发行的代币为hc,标准为500hc=1btc,我是第一个投资的人，我转入了10btc，给了我50010=5000c，因为我是初始投资，又额外给了我500hc,于是我拿到了5500hc。hero通过发行代币，拿到了10btc，它可以在交易所变现30W，获得项目发展的初始资金。我可以一直持有这5500hc，类似股票一样。hero项目发展很好，hc币后面涨到了100元/hc,则我可以卖出，拿到5500*100=55W，投资收益25/30。

这个看起来ICO非常动人，融资非常迅速，17年上半年就融资了等值26亿人民币。这块属于监管的空白，我们看看监管的声音：

  1. 坚决反对和取缔
  2. 保持观望

我们拭目以待吧！

京东的小金卡

发表于 2017-08-23

周末，刘强东率京东金融联手兴业银行共同宣布：推出中国首张互联网储蓄银行卡，取名小金卡！
这张卡有以下功能：

  1. 是借记卡
  2. 可以存钱，超过5000的部分自动转入理财享受收益
  3. 可以刷卡消费，余额不足自动从理财出，超额存入自动转入理财
  4. 不需要app，一般人去银行开卡即可

好熟悉的场景，和之前京东的小白卡类似，就是降低获客门槛，将收益最大化给到用户。银行之前完全不需要主动拉用户，现在发现用户才是最重要的。
其实这并不是先例，之前有平安的花漾卡，但是最后没有玩起来。招行也有朝朝盈，但是需要用户主动操作转入转出，而且还是限制开放。
所以，有的时候，不是你的想法不好，而是你做得不够好！
做好一件事有很多依赖：天时地利人和。
天时说的是时机，现在的大环境允许你这样玩，你可以放开玩。地利说的是优势，你有用户，有场景，有合作伙伴资源。人和说的是目标，你的团队目标明确，领导和成员共同努力！
我们看看对方利益关系。
银行：增加曝光度，获得用户存款，获得用户。
京东：平台倒流，获得用户，增加金融和消费场景！
用户：可以继续使用银行卡，存入的钱能自动增值，不需要打理。
如果存入的资金算银行存款的话，那么银行存款应该是保本的，但是这部分资金被分成了两份，小于5000的算银行存款，大于5000的部分属于理财，理财一般不保本，所以如何界定？监管如何看待这个问题？

徒然喜欢你

发表于 2017-08-18

最近徒然喜欢你更新了，看完之后发现，实在是让人产生很多以前的回忆。
每一集大概十二分钟，一般由两到三个小故事组成，有些故事说完就结束了，有些故事却在后面续集出现了。
故事的关键字：学校、纯情、同班、青梅竹马、告白、偶遇、悸动。看完，不免勾起你的情绪，让你回忆到以前学校的时光！
什么是回忆？所有的回忆都被包装过的。你现在想到的，和你之前实际处于的，是完全不一样！你现在想想你初中，能记住的有哪些？高中能记住的有哪些？大学能记住的有哪些？即使记住了，你会发现，很多都是好的回忆，坏的都被你选择性忽略或者包装了。所以，很多人都会告诉你，明天就会好的，因为那是人类进化的结果。那些一直记得没狮子老虎追逐的恐惧的野人，已经被进化抛弃了。乐观，是进化的重要手段！
喜欢回忆的人，某种程度上表明，他是一个怀旧的人，或者说对现在不满意。回忆是过去，现在和未来才是最重要的。
看完这些短片，很多人都会想起了高中时代，那时候真的很单纯啊！每天除了学习，还是学习。不知道哪一天，开始在意某个人了，眼光会不自觉往她的位置扫去，偶然对视，赶紧收回来，小鹿乱撞。每天除了学习之后，开始有点期待和他/她的偶遇，上学路上，回家路上，书店里。这大概就是很多人的青春吧！
多年以后，很多人都不会再有这样的感觉。圈子变得更小，关心的东西，开始变成我的工作是否做好，领导会不会看低我等等。每个曾经在一起的人，都在自己的圈子里面，偶尔也不会有交集。于是，只能回忆了！
说实话，我有时候比较讨厌回忆，因为回忆太美了。但是，有时候，和最爱的人一起回忆，最让人感到开心！
好好保留下你内心中最美的记忆吧！
