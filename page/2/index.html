搜索的未来

发表于 2017-10-18

Google已经宣布把搜索从文本转移到了语音和图片了，这代表着一个大方向，毕竟语音才是未来的交互方式！
文本是一维度的
图片是二纬度的
语音是三维度的
视频是四维度的
。。。。
意念是N维度的
文本应该是最简单的了，做好关键字匹配，但是也需要大量的算法和数据支持。如果上升到图片和语音，需要更大的投入和匹配。文字是冰冷的，图片包含的信息比文字多很多，语音也是一样，不止是信息还有感情！随着语音搜索感情不一样，搜索结果不一样。语音人群不一样，搜索结果不一样。语音里面需要识别关键搜索词语，每个人都可能不一样。需要识别标准语言和非标准语言，需要识别很多信息！
Google认为下一代的app交互方式就是语音，我看了它的dialogflow，包含很多词汇。可以预见，未来的服务，载体有很多，有app、智能硬件等等，所得即服务，和形式无关！所以，未来的交互，语音交互是非常重要的。
语音输入-上下文-识别意图-推送服务-完成服务-推断-重复
如何帮助现在的服务快速应用人工智能技术？无需过多的开发就能够使用？现在有很多平台提供了某种技术的快速接入，比如聚合支付，那是否可以让企业使用人工智能update自己的服务更加快捷？不管是交互方式还是服务本身？就像ping++一样很方便的接入支付。
有几点，考虑如何帮助企业解决他们的问题：
1语音交互，减少开发周期，更专注于业务本身
2人工智能应用，提高处理效率，精准度。
3信任度，解决企业对你的信任问题。

总结

如何迁移你的hexo博客？

发表于 2017-10-05

我要把我的博客迁移到新的机器
记录下步骤
1.安装git
2.安装node js
检验安装的命令：
git --version
node -v
npm -v
3.安装hexo
npm install -g hexo-cli
npm install hexo --save
检查安装是否正确
hexo -v
4.拷贝文件夹到新PC，原来设置好的配置文件和文章可以不变
5.设置GIT的身份
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
6.hexo d部署，会提示输入github的密码，输入即可

PS：
补充一个问题
最近在用hexo搭建github pages时，遇到一个问题，hexo安装没错，也能成功运行。启动也没错。
就是不能访问。。

原因是：
你的电脑端口被占用了。
hexo默认的端口是4000，如果你的电脑安装了福昕阅读器，，就是他，没错，坑爹吧！！！！
启动hexo s 的时候，用这个命令，换一个端口。
hexo s -p 5000
换成5000端口，果断可以访问

二、这句话他站在什么角度或者立场
三、这句话的背景缘由
四、此人是什么性格如何回答合适。
这样想可以培养我们的察言观色，但是这样想之前我们还需要保持心态的沉静，提高警惕心态。
慎言，多思！

昨日申城暴雨

发表于 2017-09-25

已经九月底了，然而天气还像初夏一样，阴雨绵绵。昨日夜里，被雷电惊醒，起来一看，外面雷电交加。
暴雨打在雨棚的声音，风使劲摇晃着窗外的香椿，远处电闪雷鸣，窗户在吱吱作响。
我起来靠着窗边看了下，外面雨势巨大，闪电将夜空照亮，一闪一闪的，甚是可怕。这大概就是大自然的力量吧，在自然力量面前，个体显得多么无力！坦白的说，这应该是我遇到过少数略显恐怖雷电交加的雨夜。
试想，如果这个时候在野外，在木屋里面或者山洞里面，只有一堆篝火，发着微弱的光，失去了外部的任何联系，人这个时候应该才会真正的敬仰大自然，恐惧大自然吧！最古老的图腾和崇拜，皆是从对未知的恐惧和崇拜中产生的。现在的人类，生活在被保护的境地，已经开始忘掉了对自然的恐惧，丧失了对自然的敬畏！只有偶尔的一些自然灾害，才会引起一些波动，但是很快湮没。
