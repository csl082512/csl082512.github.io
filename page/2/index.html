关于联系connection

发表于 2017-09-08

connection我称为联系，链接两个不同主体。
为了简单起见，我抽象了两个主体：人和物。
人和人的联系，称为社交，在这个领域腾讯占据领先地位。
人和物的联系，称为场景，这个领域有众多场景，淘宝、京东、点评等等。
物和物的联系，称为物联网，这个领域目前正在孵化，小米生态、百度占据。
支付是什么？支付是货币价值的流动。在人和人之间、在人和物之间、在物和物之间！
微信基于人与人的联系切入支付，水到渠成，支持收款、付款、转账等场景，同时支持部分人和物的场景。支付宝一直基于人和物的场景进行深化。微信和支付宝，两者相互竞争和渗透，微信切入越来越多人和物的场景，支付宝切入人和人的场景，结果现在已经显而易见。支付宝在人和人的场景上，完败！这是显而易见的，人和物本来就是弱关联关系。
我翻了下我的支付宝朋友记录：最近的一个朋友是一月份，是个同事，加我求敬业福，我没有。最新的朋友动态是五月，一个朋友分享了ofo红包的链接。我最新的一笔转账记录，是6月份，那个朋友说他微信没钱了，支付宝给我。社交关系差距还是蛮明显的。
我翻了下我最新的交易，早上我用支付宝付款码，付款早餐，领取了0.35元鼓励金。前一天缴了水电费，之前还有医院挂号的记录。
这样对比一下支付宝和微信：
人与人：微信绝对占优，支付宝没机会
人与物：支付宝绝对占优，微信有机会
物和物：双方都不占优，都有机会
上面就能够理解，为什么支付宝恐慌了，牌面处于劣势！微信，基于社交，主打社交，因此在人和物上，并没有像支付宝在人与人上一般发力！
因此，支付宝如果想打赢这场战争的话，就要充分利用自己的优势：深耕人与物，扩展物与物！

产品投入和产出

我们找了一家支付机构帮忙做资金托管，这是整个体系非常重要的点，真正做到T+0实时清算才是我们产品的亮点！各种服务都不错，但是有一点不好就是，价格太高了。有准入费用，有按年收取服务费。因为我们处于初期探索模式，因此收费这块我们比较谨慎，尤其是如何控制成本。
这个时候，团队的两个声音出来了：

  1. 收费太贵了，我们采用人工线下模式
  2. 继续使用服务，成本转嫁给商户
    对于1来说，很现实，采用低成本，但是这样实现，我们的产品的优势就不是那么明显了！对于2来说，采用服务，保证了我们产品特点，后面接入更多商户再扩展收费，但是有风险！
    我是坚持2的，如果最后没有替代品，我坚持使用的，产品的特点才是最重要的，最吸引用户的。

    钱该花的时候就要花，该投入就投入！
    最后，我们找到了替代产品，效果一样，但是费用超低，我们肯定会用这个产品。
    但是，过程中，我大概知道了每个人的思考方式和产品感觉！

聊聊ICO

发表于 2017-09-01

这次，我们聊下ICO吧。
ICO(initial coin offering)首次代币发行，提供了一种新的融资方式。
传统的融资方式：

  1. 私人注资
  2. 天使投资
  3. 银行借款
  4. 发行债券
  5. IPO

IPO是很多公司都想参与的，IPO之后发行的股份，流动性比较好，也能融到大量的资金。
但是，这些都有个前提，那就是准入要求，或多或少需要你提供良好的证明，注册资本、连续几年的营业收入，等等。
ICO就不一样，经常出现的场景是，只凭借一份几十页的白皮书，就能融资几个亿，略显疯狂。
我们来看看ICO的流程：

  1. 推出项目介绍网站，发布项目白皮书
  2. 开启ICO流程
  3. 结束ICO流程
  4. 披露信息

总体上，目前ICO的标的物可以分成以下几个种类：

  1. 虚拟商品类
  2. 收益权凭证类
  3. 基金份额类
  4. 公司股权类

这是一个比较快速的过程，很快就能够融资。这里面有几个比较重要的概念：
项目白皮书
项目白皮书是ICO项目的灵魂，一般阐述以下内容：

  • 项目简介
  • 市场概述
  • 任务和愿景
  • 我们提供的解决方案
  • 运作方式
  • 竞品分析
  • 发展规划
  • 管理规划
  • 代币介绍和发行计划
  • 代币的价值和分成计划
  • 资金用途
  • 团队和投资者

代币
代币一般是和现有的数字货币挂钩,数字货币一般是比较主流的，主要是比特币、莱特币、以太币、瑞波币等。设置一个比例，比如：500代币=1比特币。
代币一般基于以太坊的智能合约发行，也有一些平台自建。用户转入比特币，获取对应的代币。为了鼓励人参与，最先转入的人能够获得一些奖励。拿到了代币，象征着权益，后续可以转移交易。可以对接交易所，方便进行买卖。代币的价值是和项目的价值挂钩的，比如，如果项目发展很好，则估值很高，代币的交易金额就会快速上涨。或者，代币代表着某种权益，使用的权利，类似ripple币，则随着使用的人变多，价格也增加。因此，ICO更多可以称为产品众筹，结果取决于项目的表现。

举个例子：
我现在手头有10btc，按照现在的行情来看，我在OKCoin交易所可以变现310=30W。
目前，有个ICO项目，叫做hero，发行的代币为hc,标准为500hc=1btc,我是第一个投资的人，我转入了10btc，给了我50010=5000c，因为我是初始投资，又额外给了我500hc,于是我拿到了5500hc。hero通过发行代币，拿到了10btc，它可以在交易所变现30W，获得项目发展的初始资金。我可以一直持有这5500hc，类似股票一样。hero项目发展很好，hc币后面涨到了100元/hc,则我可以卖出，拿到5500*100=55W，投资收益25/30。

这个看起来ICO非常动人，融资非常迅速，17年上半年就融资了等值26亿人民币。这块属于监管的空白，我们看看监管的声音：

  1. 坚决反对和取缔
  2. 保持观望

我们拭目以待吧！

鲁尼宣布退出国家队！
早上看到的这个消息，突然心中一震，有种莫名的失落感！如果说转会埃弗顿是为了打主力进国家队，那么这个决定就把一切都推翻了！
我们重新看鲁尼的转会：
Rooney和卢卡库互换东家，曼联掏出了7500万英镑，鲁尼的转会费没有透露，应该很少或者免费。曼联获得了卢卡库，埃弗顿获得了鲁尼还有7500万英镑，鲁尼回到了从小就爱的球队，获得主力，薪水不变！三方共赢的局面。
鲁尼应该是截糊切尔西拿下卢卡库的关键，如此鲁尼，怎能不爱！
现在英超，我会看三支球队比赛，曼联、埃弗顿、西汉姆，不为别的，曼联是挚爱，鲁尼在埃弗顿，小豌豆在铁锤帮！
鲁尼对埃弗顿是真爱，绝对是，毕竟是从小就追随，而且是成名的地方！所以，退出国家队就很好解释了：1.让位给年轻人 2.专注埃弗顿 3.反正自己能带给英格兰的不多了，激流勇退未尝不可。
鲁尼对曼联也是真爱，13年不是玩的，曼联历史最佳射手！临走之前，替补不闹，勤勤恳恳。那些说鲁尼的人，闭嘴吧！
说鲁尼和C罗的对比，这就是搞笑。人各有志，每个人都有自己的想法。英格兰人，在自己国家最大的球队，无可厚非！为了团队，无可厚非！
C罗就像二次创业获得巨大成功的关键人物，转投另一个公司，后面获得了巨大的成功。鲁尼就是在C罗离开之后，一个人带领原来的公司踽踽前行的人！如此而言，对曼联来说，谁是功勋？无需多言！

1.经验 2.核心 3.稳定

思考的危机？

发表于 2017-08-24

习惯挑战的人，总会不安于现状。现状一般都是一成不变的，很难有些东西被打破！就好像围墙，很多人都困在里面，羡慕外面的人，但是却不敢出去。像进击的巨人一样，总是担心城墙外的庞然大物撕裂自己！而选择出去的人，一部分刚刚出了城墙，就被巨人撕裂了，另一部分目睹的人立刻退回去了，还有一部分人继续朝着城外腹地前行。最终的结局，往往是在外面又创造了一个新的围墙，只是可能更低一点！
最终进化改变的，永远是往外面走的人。
