战狼II想到的

发表于 2017-08-05

周六下雨，暴雨，好久没有看电影了，翻了翻最近上映的电影，战狼II大家评价不错，打算去看看。
十天居然有20亿的票房，我想这个市场已经疯了，毕竟13亿的中国市场还是非常庞大的。战狼／湄公河行动，这两部电影，我归为一类，因为题材都比较类似。
全场两个多小时，视觉效果还是挺好的，2亿的预算看来还是花的比较实在，不像某些用明星堆叠出来的电影，片酬占据了预算的大部分，效果也就呵呵了。
看完之后，我重点说几句：

  1. 这部片子应该被删掉了不少，不过能放出来说明有些进步了
  2. 涉及外交、军事行动、个人英雄主义，这类的题材貌似国内还是比较少有的
  3. 带动了所有人的国家情怀，触及了大家的情绪
  4. 电影本身没有任何情节，就是视觉效果，不需要脑细胞，看就可以了，老少咸宜

要说类似的动作片，背景很重要，现在社会的背景下很容易引起共鸣。如果说，背景定在40年代，70年代，则和观众太脱节了。现在，很容易大家就在国外学习工作旅游，难免会遇到很多想不到的事情。首先，背景定位，就能够让观众接受了。然后，没有烧脑的剧情，就是打斗场景，绚丽的效果，让你觉得很爽。抛去情节空洞，完全的好莱坞大片套路，再加上渲染名族主义，票房大卖也就不足为奇了。
相比之下，看同期的两部电影《建军大业》《悟空传》，只能看剧情了，找了小生扮演，或者实力派，但是终究还是抵挡不住。说白了，就是没有和观众产生共鸣。用户不会为了看电影而去看电影，不会为了建军节而必须去看建军大业，不会为了看过悟空传的小说而去看悟空传。但是，一旦产生了共鸣，就不一样了。先不说引入的好莱坞大片，就看国产的影片。我印象中，近几年的票房，《大圣归来》《捉妖记》《泰囧》基本都是类似的套路。大圣归来告诉大家国产的3D动画也可以不错，泰囧完全是带起来了一堆表情，捉妖记引入了卖萌的胡巴和动画。先不说质量如何，只看市场定位的话，确实是挺不错的。
请记住一句话，票房再好的电影，基本都不会获得电影奖项。哪年的奥斯卡是给你看过的电影的？票房是群众的选择，奖项则是艺术家的选择，群众永远不可能成为主流。
在现在膨胀的市场，借助互联网的传播，一切变化得太快。可以预见，以后的钱将会越来越不值钱，跟风和效仿将会越来越多。战狼有这么好的票房，马上就会有其他类似的出来。更多好莱坞式的电影将会越来越多，目的就是为了赚钱。不信的话，看看吧。
总有一些人默默的坚守，坚守自己的风格，坚守拍那些让人看不懂的电影，那种电影属于艺术，不属于商业。
艺术永远只属于少数人，属于大众的艺术，就是商业而已。
同样，足球的世界也开始变了。一年前pogba的一亿的转会费，大家还说高，突然今年随便买个人都要四千万欧元，然而Neymar的转会费居然到了2.2亿欧元，准确说是解约金。据说，全部费用加完，需要5亿欧元，这实在crazy。这都是巨大的泡沫，狠狠冲刷了历史，冲刷着摇摇欲坠的忠诚。商人只看利益，亘古不变的道理。
我想用不了多久，战狼II会超过美人鱼的33亿票房，因为它找了一个大家无法拒绝的理由，而且很简单的理由。
让我们拭目以待！

网易云音乐的splash页面

发表于 2017-08-04

最近打开网易云音乐，发现splash页面有点改变。
splash页面主要有两个目的：

  1. 为加载数据赢得时间
  2. 广告和宣传推广

如果是本地应用，其实1是不需要的，所以你看到原生应用基本都没有，但是现在很多都加了splash，主要还是广告。
大部分的splash页面都好挫，鉴定完毕。要么是一张图片，要么是一个GIF，或者是跳动的广告，五颜六色，实在是让人觉得乱入了。但这次，网易云音乐的splash让我眼前一亮。
主要用了动画引导，结合灯光的变化效果，打造了一个循序渐进的效果。简单说，就像你回家的时候，家里黑乎乎的，你全部按下了所有的开关，然后灯光按次序一点点打开，先是廊灯，然后壁灯，然后橱柜的灯光，再是顶部大厅的灯，最后整个温暖的家展现在你面前。
既然是引导，就要有次序，有带入感，只是现在很多人都不看重，反而觉得这是简单的投放广告的赚钱手段。
坦白的说，除非你的内容特别有趣，否则我不会强迫自己去忍受杂乱、广告乱入的页面。一般，比如你的内容是最新的网络小说章节、或者就是必须信息的，很难搜到的，用户才会勉强留下来。一旦有更好一点的选择，抛弃你毫不犹豫。
设想一下，是否可以针对splash制作一些广告分发平台？只做精致的广告，有内容，有次序的广告。
广告内容的格式：

  1. 图片
  2. GIF动画
  3. 微视频

体验要好，比如能够针对设备分辨率匹配不同大小的图片等。提供一些广告插入解决方案，可以快速接入，而且体验不错。比如按照次数，触及一定的次数，需要看一些广告才可以，我比较推荐像threes那样的广告模式。
初始给你15次可玩次数，中间不会有任何广告，屏幕也不会有，次数减到0就开始提醒你需要观看广告了。

百度推广也真的是无敌了

发表于 2017-08-03

昨天刚刚在阿里云上面注册了一个域名，就是我现在的这个busyram
今天早上就接到电话，问我是不是注册了一个域名，我没正面回答，后面就开始了，问我要不要推广
呵呵了，我就不知道那帮人怎么来的，后来仔细了解了一下，发现以下：
在阿里云注册的域名，不会自动开启隐私保护，导致在whois里面一查就能够查到了，包括地址，邮箱，手机号，吓得我赶紧关掉了
百度是怎么拿到的呢？以我做产品经理的经验，这是运营推广必须的，数据来源：爬虫，买，运营商合作。拿到数据立刻电话咨询，直接面对潜在用户。
我们鄙视也罢，不屑也罢，这确实是别人的手段。从推广的角度来说，这样确实是比较精确的：
你注册了一个域名–你要建网站–你可能需要推广你的网站–我有推广和搜索竞价系统。
这一连串，从产品推广来说，确实很好，也是很容易想到的方案。我突然想起来了一个case：
有个人告诉我，当时他们是怎么推广他们产品的，竞争对手的注册登陆有个缺陷，输入邮箱地址会告诉你是否已经注册，正好QQ号码是有序的，于是他们写了小程序，轮询QQ号码对应的邮箱，然后发推广邮件，据说效果还不错。
作为用户，用两种情况：
第一类：如果用户不想接受推广，不想建站，用户直接拒绝
第二类：如果用户正好想推广，而且百度在国内的压倒性优势，用户可能会选择合作
很简单，我现在属于第一类，而我们公司属于第二类，会主动推广，买关键字。统计过，通过官网搜索的排名，客户量还是挺可观的。自然，钱也是要花的。

补习一下HEXO和MD文件的操作

发表于 2017-08-03

欲善其事，先利其器

hexo操作

HEXO的几个操作明令，简单整理一下：

新建网站

hexo init [folder]
新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

新建文章

hexo new [layout] ‘title’
新建一篇文章，可设置标题。
文章的文件名和标题一致，因此我选择在title增加时间，这样在文章文件中有，便于检索，写文章的时候把title的时间删掉即可.(可以在站点配置文件自动加上去)
补充一点：
layout是布局的，常用有draft，post，post的时候可以不用指定。
hexo new draft ‘标题’
创建了一个文章的草稿
hexo publish draft ‘标题’
将草稿移动到发布文件中

发布草稿

hexo publish [layout] ‘title’
通过 publish 命令将草稿移动到 source/_posts 文件夹,和new很类似

生成静态网页文件

hexo generate 可简写为：hexo g

启动本地服务器

hexo server
启动服务器。默认情况下，访问网址为： http://localhost:4000/
发布之前可以预览看下效果

部署网站

hexo deploy 可以简写为：hexo d

清除缓存和静态文件

hexo clean
一般在更换主题的时候用，需要重新清理缓存的时候用，一般并不需要

busyram代表什么？

发表于 2017-08-02

最开始，我最好能够说明下为什么我的域名叫做busyram。这是一个很重要的问题。
ram是什么？

随机存取存储器（random access memory，RAM）又称作“随机存储器”，是与CPU直接交换数据的内部存储器，也叫主存(内存)。它可以随时读写，而且速度很快，通常作为操作系统或其他正在运行中的程序的临时数据存储媒介。

ram是智能设备里面最重要的东西，这样类比吧，CPU好似心脏，ram类似大脑临时存储记忆的海马体，rom就相当于长期记忆的大脑皮层。
busyram的意思，重点在busy和ram上，你的记忆、思维需要动起来。
没有用的ram，再多也没用，因为堆叠出来的无用东西就是浪费。如果你用iPhone的话，iPhone6还是1G的ram，此时其它的Android手机都是2G甚至4G，但是iPhone6并没有给人卡顿的感觉，Android就会让人觉得。
所以，我期望能够做一个busyram！
